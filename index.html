Hachiko

Windows で動く通知ハブです。ローカル API とファイル監視を使って、 Discord、Slack、Telegram、Microsoft Teams へ通知を送れます。

配布ファイルの保管に GitHub Releases を使っていても、 利用者の正面入口はこの公式ページにまとめる想定です。

Windows 版をダウンロード GitHubで見る 更新情報を見る

対応環境: Windows 10 / 11
配布形式: Win32 installer HachikoSetup-0.1.2.exe

ダウンロードボタンは installer 本体の直リンクです。 GitHub の一覧ページを見たい場合は「GitHubで見る」を使ってください。

配布方針

Hachiko は Windows Service を使うため、Store ではなく Win32 installer を前提にしています。 更新確認はアプリ内からこの公式ページを開く形にして、公開説明、更新内容、ダウンロード導線を一か所へ集めます。

利用者に見せる URL はこの HTML ページを基本にして、GitHub Releases は裏側の保管先として使う形でも問題ありません。
