Hachiko

Hachiko is a Windows notification hub. It can send alerts to Discord, Slack, Telegram, and Microsoft Teams through a local API and file watching.

Even if GitHub Releases is used as the storage location for binaries, this page is intended to be the main public entry point for users.

Distribution

Hachiko uses a Windows Service, so it currently ships as a Win32 installer rather than a Store app. The in-app update flow opens this official page so release notes and download guidance stay in one place.

GitHub Releases can remain the backend storage for binaries while this HTML page stays the user-facing homepage.
